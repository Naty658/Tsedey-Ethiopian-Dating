# <Tsedey Ethiopian Dating>

A modern Android dating app built with **React Native (Expo)** and **Firebase (Auth + Firestore)**. Published on **Google Play** with **100+ installs** and **50+ registered users**.

## Highlights
- Swipe & match flow
- Real-time chat
- Online status (presence)
- Filters (age, location, preferences)
- Profile privacy (hide profile)
- Safety tools: block, report
- Chat controls: delete chats
- UX controls: reset swipes

## Tech Stack
- React Native (Expo)
- Firebase Authentication
- Cloud Firestore (React Native Firebase SDK)
